# Hamrohio — Landing Page

O'zbekistonning birinchi aqlli desk roboti uchun landing page. Next.js 14 (App Router), Tailwind CSS, Framer Motion.

## Ishga tushirish

```bash
npm install
npm run dev
```

Brauzerda [http://localhost:3000](http://localhost:3000) oching.

## Deploy (Vercel)

```bash
npm run build
```

Vercel'ga repo ulang yoki `vercel` CLI orqali deploy qiling.
